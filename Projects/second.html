CloudFrontiers
Project Lead/Full-Stack Developer

CloudFrontiers is a Proof of Concept website that was developed as the final project for my Spring 2024 Web Science Class.

It provides ready-to-go Market Analysis for various cloud-based products, including details such as the pricing and what service it is included in.

The site allows users to look up various type of products (NoSQL Databases, Cloud Containers, etc) and view various offering from various vendors, and allow them to view any 2 offerings compared side-by-side. The site then offers users to export the generated data as either a JSON or CSV file.
